Volta

My Project


نبذة عن المشروع:

Volta هو نظام ذكي لإدارة الأحمال الشمسية صُمم لتحليل إنتاج واستهلاك الطاقة والتحكم الفعّال بالأجهزة المنزلية.
يجمع المشروع بين مكونات برمجية ومادية لربط الخادم الخلفي (Backend) بالأجهزة الحقلية مثل الإنفرترات والبطاريات والمقابس الذكية، مع توفير آليات لإدارة الأحمال بناءً على توافر الطاقة.

تحليل النظام وجمع المتطلبات:

- تحليل متطلبات العميل والمتخصصين التقنيين لتصميم نظام يلبي احتياجات إدارة الطاقة الذكية.
- تصميم المخططات التقنية مثل ERD, Use Case, Sequence Diagrams لتنظيم تدفق البيانات والعمليات.
- العمل بمنهجية Agile عبر سبرنتات متكررة لضمان تحسين مستمر وتسليم ميزات تدريجية.

    التقنيات المستخدمة

  • - Backend: Laravel, PHP, MySQL, Redis, WebSockets
  • - تواصل الـ API: RESTful APIs, MQTT
  • - الأمان: Laravel Sanctum، نظام صلاحيات (RBAC)، نقل مشفّر للبيانات
  • - تطبيق الموبايل: Flutter (من فريق العمل)
  • - لوحة الإدارة: HTML, CSS, JS (من فريق العمل)
  • - جهاز SLMT: C/Arduino، UART، RS-485 (Modbus) (جزء ميداني)


دوري – مطور Backend ومحلل نظم

1. تطوير الواجهة الخلفية (Backend):
- تصميم وتطوير واجهات برمجة تطبيقات (RESTful APIs) باستخدام Laravel لربط النظام بواجهات الإدارة، التطبيقات، والأجهزة الميدانية.
- بناء نظام إدارة مستخدمين وأدوار يدعم صلاحيات المدير، الخبير الفني، والمستخدم النهائي مع سياسات أمان قوية.
- تطوير نظام جدولة ذكي يتحكم في تشغيل الأجهزة بناءً على توقعات إنتاج الطاقة والاستهلاك.
- دمج WebSockets وعمارة معتمدة على الأحداث (Event-Driven) لإرسال تنبيهات لحظية عند تغيّر الفولت أو الاستهلاك.
- تصميم وتحسين قاعدة بيانات MySQL لضمان استعلامات سريعة وفعّالة باستخدام Eloquent ORM.
- تنفيذ تكامل مع وحدات التحكم المنزلية (SLMT) عبر بروتوكولات HTTP وMQTT لإرسال الأوامر واستقبال البيانات الحية.
- بناء لوحة مراقبة لتسجيل ومتابعة طلبات الـ API وتحليل أداء النظام.

2. هندسة النظام وتحليل المتطلبات:
- جمع وتحليل متطلبات العميل والفريق الفني وكتابة مستندات المتطلبات (SRS).
- تصميم مخططات هندسية (ERD، Use Case، Sequence) لتوضيح بنية البيانات وتدفق الأحداث.
- إعداد حالات اختبار (Test Cases) للتحقق من منطق العمل وصحة الـ APIs في مختلف السيناريوهات.

وحدة SLMT – وحدة إدارة الأحمال الذكية

وحدة التحكم الذكية (SLMT) هي واجهة ميدانية تقوم بجمع بيانات الجهد والتيار والاستهلاك وإرسالها إلى الخادم عبر MQTT/HTTP، كما تستقبل أوامر التشغيل/الإيقاف لإدارة الأحمال وفق سياسات النظام.
